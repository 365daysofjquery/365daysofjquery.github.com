Much the same as adding text in jQuery you can also add HTML like this:

	$('body').html('<h1>Here Is Some HTML</h1>);


Similarly when using `'` you'll need to add a `\` before it.

	$('body').html('<h1>Here\'s Some HTML<h1>');
